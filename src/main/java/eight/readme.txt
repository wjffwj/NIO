java 的编解码

ASCII:（美国信息交换标准代码）用7位表示1个字符 一共 128种字符

ISO-8859-1: 8位表示1个字符  即用1个字节表示1个字符 一共256个字符 完全兼容ASCII码

gb2312: 用2个字节表示1个汉字  ，但是不支持生僻字

gbk:  对gb2312扩展

gb18030 :最完整汉字表示形式

big5: 繁体中文的表示形式


unicode ：全世界的所有字符 采用了2个字节表示1个字符  16位  256*256个字符  但是不适用于存储

UTF-8   unicode translation format   是一种存储格式

unicode是1种编码方式   而UTF是存储方式

1一般来说utf8通过3个字节表示中文
